# 🎨 Banana Editor🍌

Bana Editor is an AI-powered image editing platform that allows users to generate, enhance, and transform images using simple prompts and intelligent editing tools.

Originally inspired from **Nano Bana**, the project evolved into a more powerful and feature-rich editor — now known as **Bana Editor**.

## 🚀 Features

### 🧠 AI Image Generation

- Generate images using natural language prompts
- Powered by advanced generative AI models

---

### 🎭 AI Filters

Apply creative styles to your images instantly:

- Toonify
- Ghibli Style
- Cyberpunk
- Oil Painting

---

### 📐 AI Expansion (Aspect Ratio Resize)

Expand images intelligently into different aspect ratios:

- Instagram Feed **[1:1]**
- Youtube / video **[16:9]**
- Classic Camera **[4:3]**
- DSLR / Print **[3:2]**
- Ultrawide **[21:9]**
- Tiktok / Reels **[9:16]**
- Insta Portrait **[2:3]**
- Pinterest **[4:5]**

---

### ✂️ AI Editing Tools

- Remove Background
- AI Refreshment (enhance & regenerate image details)

---

## 🛠️ Tech Stack

- **Frontend & Framework:** Next.js
- **State Management:** Zustand
- **AI Model:** Gemini 3 Pro
- **Image Generation:** Google GenAI APIs

## ⚙️ How It Works

1. Enter a prompt or upload an image
2. Choose a feature (generate, filter, expand, edit)
3. AI processes the request in real-time
4. Get enhanced or newly generated output instantly
